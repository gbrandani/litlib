‘Construction of this table involved three steps. 1) The sequences of all known a//I barrel proteins were retrieved from a combined database (GenBank release 81, Swiss-Prot release 27, and NBRF-PIR release 39) using the program GENEMAN in the DNASTAR program suite. For each individual protein, all available sequences were aligned with each other using the program ALIGN in the NBRF-PIR program suite. The Dayhoff mutation data matrix with a break penalty of 20 was used as the scoring matrix. Each sequence was then randomized 100 times and the jumbled sequences were aligned using the same matrix. These jumbles gave an average value and a standard deviation. Alignment scores are reported as the number of standard deviations above the jumbled average. 2) The variation of sequences within individual proteins was then examined. For 19 proteins the variation was small and one sequence could be considered characteristic of all known sequences. The characteristic sequence is listed underneath the name of the protein. For the remaining proteins, it was obvious that more than one sequence was necessary. 3) After all characteristic a/fl barrel sequences had been identified, the entire sequence database was then searched, with each sequence using the similarity search algorithm implemented in the program GENEMAN. search parameters were ktuple 2, window = 64, gap penalty = 4, and threshold = 20%. Finally, all possible a//I barrel proteins identified using this fast search were then aligned using the more definitive ALIGN method described above.
